## **Vue3-demoプロジェクトの起動方法**

- ### `/vue3-demo`に移動

```sh
cd vue3-demo
```

- ### パッケージをインストール

```sh
npm install
```

- ### 起動

```sh
npm run dev
```

- ### 画面を確認

http://localhost:3001/
